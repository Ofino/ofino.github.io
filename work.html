Proposal Funding Performance Analysis

Tools: Power BI (DAX), SQL, Data Modeling, KPI Framework Design, Financial Aggregation Logic

Results: Built a centralized dashboard that highlighted differences in approval rates across categories and created a reusable framework for tracking funding performance.

Business Context

Limited visibility into how funding requests converted into approvals.

While supporting improvements to the NEAR Treasury platform’s data visibility, I noticed that funding approvals varied across categories but weren’t easy to evaluate in a structured way.

Proposal data existed, but there wasn’t a simple way to see:

  • Which categories had strong approval rates
  • Where requested funding wasn’t converting
  • How capital was distributed across programs

This project was created to organize that information into something clear and measurable.

Objective

Evaluating how effectively requested funding translated into approved capital.

The goal was to evaluate how effectively requested funding converted into approved capital.

Specifically, I wanted to understand:

  • Which funding categories demonstrate the strongest approval performance?
  • Where approval rates are underperforming relative to requested capital?
  • How total requested funding compares to approved outcomes across segments?

The focus was clarity — turning raw proposal data into something easier to interpret.

Data & Approach

Structuring proposal data into consistent, comparable performance metrics.

I structured proposal-level data in Power BI and defined a small set of consistent metrics:

  • Total Requested
  • Total Approved
  • Approval Rate %
  • Funding Utilization %

This created a simple framework for comparing categories side by side.

Key Insights

Identifying approval patterns and funding concentration trends.

Key Insights:

A few patterns became clear:

  • Approval rates varied significantly by category.
  • Some categories requested large amounts of funding but had lower approval efficiency.
  • A small number of categories accounted for most requested capital, creating concentration risk.

These patterns suggested a potential misalignment between capital allocation and approval performance.

Outcome & Impact

Creating clearer insight into funding performance and allocation efficiency.

This dashboard organized proposal data into a clear reporting framework that made funding performance easier to evaluate. It demonstrates how structured KPI modeling can improve visibility into allocation efficiency and support more informed decision-making. The framework can be reused to track funding performance over time.
